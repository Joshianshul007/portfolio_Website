# Anshuman Joshi — 3D Developer Portfolio

Welcome to my personal portfolio repository! This is a high-performance, fully responsive 3D web experience built to showcase my transition from full-stack MERN mechanics into high-fidelity front-end engineering, interactive UI/UX, and generative front-end architecture.

## 🚀 The Architecture & Tech Stack

This project leverages hardware acceleration to deliver a truly cinematic web experience.

*   **Core:** React.js, TypeScript, Vite
*   **3D Engine & Physics:** Three.js (`@react-three/fiber`, `@react-three/drei`), Rapier Physics Engine
*   **Animation Array:** GSAP (ScrollTrigger & ScrollSmoother) for heavy scroll hijacking and seamless reveals.
*   **Styling:** Raw highly-optimized Vanilla CSS prioritizing GPU-accelerated transforms (`translate3d`, `opacity`, `rotateX/Y`).

## ⚙️ Key UI Features

*   **Absolute Crossfade Engine:** A custom-built vertical stacking carousel that utilizes spatial depth curves to instantly crossfade projects without triggering layout thrashing.
*   **Physical 3D Glass Portals:** Project thumbnails wrapped in a native WebGL-style CSS 3D wrapper that mathematically responds to cursor velocity—rotating and casting dynamic light glares upon hover.
*   **Dynamic Tech Tree:** A pseudo-randomized organic GSAP-powered floating map displaying technologies (React, Next.js, Node, Firebase, etc.) reacting naturally on scroll boundaries.
*   **Loading Gateway:** A fast-booting, custom marquee loading screen that gates the application until the massive 3D models and textures are successfully decompressed and cached in the browser.

## 🛠️ Local Development

Want to spin this up locally?

```bash
# Clone this repository
git clone https://github.com/Joshianshul007/<your-repo-name>.git

# Install dependencies
npm install

# Start the high-performance local dev server
npm run dev
```

Build for production:
```bash
npm run build
```
